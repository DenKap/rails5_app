# rails 5.0.2 app

 Sample rails app for managing and display data.

* Ruby 2.4.0

* Rails 5.0.2

# Basic information:

* Login through omniauth: Facebook, Twitter

* Google Maps API - geocoder gem

* File upload with paperclip gem

* Bootstrap (basic views) for frontend

* jQuery

# Testing:

* RSpec

* Factrory Girl

* Shoulda Matchers

All the rest is inside.
